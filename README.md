# Intelligent Job Recommendation System

An Intelligent Job Recommendation System developed using **Python, NLP, and Machine Learning** to recommend relevant jobs based on user skills and preferences.

## 🚀 Features

* Data preprocessing and cleaning
* TF-IDF for text vectorization
* PCA for dimensionality reduction
* KNN with Cosine Similarity for job recommendations
* Top-5 personalized job recommendations
* Data visualization and analytics

## 🛠️ Technologies Used

* Python
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
* Jupyter Notebook

## 📊 Dataset

Kaggle Data Science Job Postings Dataset (12,000+ job records).
